## 使用列表的一些注意事项

如果在单一列表项中包含了多个段落，为了保证渲染正常，*与段落首字母之间必须保留四个空格

*    段落一
     段落二

+    段落一
     段落二

-    段落一
     段落二

另外，如果在列表中加入区块引用，区域引用符也需要缩进4个空格

*    >区块引用一
     >区块引用二

+    >区块引用一
     >区块引用二

-    >区块引用一
     >区块引用二
